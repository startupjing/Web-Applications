  - post, edit, delete, search news and stories

  - guest users
  - search function
  - save story and category to favorites, follow other users
  - update personal information
  - security desing
    + encrypted password
    + safe from SQL injection attacks
    + CSRF tokens are passed
    + FIEO philosophy
